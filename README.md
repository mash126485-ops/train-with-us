v# Gym Instructor Business Platform
This repository contains the customer-facing application structure for a professional fitness coaching business.

## Project Structure
- `index.html`: The main landing page with niche categorization.
- `style.css`: Modern dark-mode UI styling.
- `intake.md`: Client onboarding questions.
- `waiver.md`: Legal and medical clearance templates.

## How to Deploy
1. Create a new GitHub Repository.
2. Upload these files.
3. Go to **Settings > Pages**.
4. Select "Deploy from a branch" and choose **Main**.
5. Your website will be live at `https://yourusername.github.io/reponame`.
